# FastVK Examples

All examples are runnable. Set `VK_TOKEN` and `VK_GROUP_ID` before running.

```console
$ export VK_TOKEN=vk1.a.YOUR_TOKEN
$ export VK_GROUP_ID=123456789
$ python examples/basic/echo_bot.py
```

## Categories

- **basic** — echo bot, commands, user info
- **fsm** — multi-step forms, quizzes
- **keyboard** — regular and inline keyboards, callback handlers
- **filters** — built-in and custom filters
- **middleware** — logging, rate limiting
- **routers** — splitting handlers across modules
- **events** — group_join, group_leave, wall posts
- **advanced** — raw API calls, full showcase

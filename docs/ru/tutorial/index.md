# Туториал

Туториал покрывает все основные возможности FastVK шаг за шагом.

- [Первые шаги](first-steps.md) — установка, первый бот, запуск.
- [Хэндлеры](handlers.md) — `@bot.message()`, `@bot.callback()`, события группы.
- [Фильтры](filters.md) — `Command`, `Text`, `F`, `StateFilter`, regexp.
- [Сообщение](message.md) — `answer()`, медиа, редактирование, закрепление.
- [Клавиатура](keyboard.md) — `Button`, `Keyboard`, inline и обычная.
- [FSM](fsm.md) — состояния, `FSMContext`, хранилища.
- [Роутеры](routers.md) — разбивка бота по файлам.
- [Middleware](middleware.md) — перехват событий.
- [Lifespan](lifespan.md) — код при старте и остановке.
- [Обработка ошибок](exceptions.md) — `@bot.exception_handler()`.
